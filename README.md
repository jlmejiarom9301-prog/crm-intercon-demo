# Demo Ejecutiva CRM Inter-Con V2

Demo interactiva del CRM Comercial Inter-Con. Una presentación ejecutiva que muestra la transformación de un captador de leads a un sistema operativo comercial integrado.

## 🚀 Uso Rápido

1. Abre `index.html` en Chrome, Edge o Firefox
2. Navega por el menú lateral
3. En "Actual vs objetivo", presiona "Animar comparación" para ver la animación

## 📋 Secciones

- **Resumen ejecutivo** - Propuesta de valor en 4 pilares (Captura, Control, Automatización, Medición)
- **Actual vs objetivo** - Comparación animada del flujo actual vs flujo objetivo
- **Flujo del lead** - Recorrido inteligente del lead (6 etapas interactivas)
- **Ecosistema** - Arquitectura modular con integraciones (Apollo, Tapni, Chatbot, Google Ads, Metricool, n8n, Outlook, Power BI, IA)
- **Módulos comerciales** - Segmentación por mercado (Protección física, Seguridad electrónica, Sectores estratégicos)
- **Vs HubSpot** - Tabla comparativa de propuesta de valor

## 💻 Tecnología

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsivo con animaciones
- **Vanilla JavaScript** - Interactividad sin dependencias

## ✨ Características

- ✅ Navegación fluida entre secciones
